## ざっくり概要
- RailsをAPI専用で生成
- 毎朝7時に雨予報メッセージを送ってくれるようにheroku schedulerを設定
- 特定の単語を含んだメッセージを送ると、それに応じて返信してくれる(画像参照↓)
